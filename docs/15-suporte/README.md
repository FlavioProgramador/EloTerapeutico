## Coleta segura de diagnóstico

Inclua:

- ambiente e SHA;
- comando/rota;
- horário e timezone;
- status HTTP;
- tipo de exceção;
- logs redigidos;
- passos para reproduzir com dados fictícios.

Nunca inclua `.env`, Authorization, CPF, prontuário, arquivos de paciente, API key ou connection string.
