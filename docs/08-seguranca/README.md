## Classificação

- **Implementado:** verificável no código e testes;
- **Operacional:** depende de infraestrutura/processo;
- **Pendente:** não existe ou não foi comprovado;
- **Bloqueador:** deve ser resolvido antes de dados reais conforme o risco.
