## Resumo

O Elo Terapêutico é uma aplicação web para centralizar atividades clínicas e administrativas de profissionais de saúde. O sistema possui frontend Next.js e API Django, com recursos para pacientes, prontuários, agenda, financeiro, documentos, formulários, relatórios, assinaturas e administração.

A aplicação está em desenvolvimento e ainda requer decisões operacionais, validação jurídica e hardening antes de uso com dados reais em produção.
