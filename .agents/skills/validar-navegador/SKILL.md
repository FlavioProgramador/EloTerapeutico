# Validar no navegador
1. Inicie os serviços locais pelos comandos documentados.
2. Teste carregamento, fluxo principal, validação, erro de API, vazio, teclado, foco e responsividade.
3. Verifique console e rede em celular, tablet e desktop.
4. Use apenas dados fictícios em screenshots.
5. Registre problemas com página, passos e severidade.
6. Após correções autorizadas, repita o fluxo e gere um walkthrough curto.
